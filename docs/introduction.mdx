## Getting Started in 3 Steps

<Steps>
  <Step title="Install & Configure">
    Run `npm install Swiftgum` (or similar) and set your environment variables (`Swiftgum_URL`, `Swiftgum_TOKEN`).
  </Step>

  <Step title="Manage Integrations">
    In the Swiftgum dashboard, set up OAuth apps (Google, Notion, etc.) and grant end users the ability to connect their accounts.
  </Step>

  <Step title="Leverage in AI Flows">
    Use our built-in connectors and retrieval pipeline to incorporate real-time documents into your LLM or RAG workflows.
  </Step>
</Steps>
